### 目录结构

每门实验课独立维护目录，如 `x-ai/`、`x-img/`，包含：

- `md-*.md`：课程文档
- `*.ipynb`：Jupyter 笔记本作业
- `scripts/`：前端交互脚本
- `styles/`：页面样式
- `img/`：课程资源图片
